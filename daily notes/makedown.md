~~~md
📘 Markdown 문법 정리 (실전 치트시트)

마크다운(.md) 문서를 작성할 때 자주 사용하는 문법을
나중에 참고하기 위해 정리한 문서입니다.


아래 내용은 문법이 적용되지 않고 그대로 표시됩니다.

==================================================

1️⃣ 제목 (Heading)

# 제목 1 (가장 큼)
## 제목 2
### 제목 3
#### 제목 4

==================================================

2️⃣ 줄 바꿈 / 문단

✅ 문단 나누기 (권장)

문장 A

문장 B

⚠️ 줄만 바꾸기 (같은 문단)

문장 A␠␠
문장 B

= 스페이스 2번 + 엔터

==================================================

3️⃣ 강조 / 꾸밈

**굵게**
*기울임*
***굵고 기울임***
~~취소선~~
`인라인 코드`

==================================================

4️⃣ 목록 (List)

● 순서 없는 목록
- 항목 1
- 항목 2
  - 하위 항목

● 순서 있는 목록
1. 첫 번째
2. 두 번째
3. 세 번째

==================================================

5️⃣ 체크리스트 (Todo)

- [ ] 할 일
- [x] 완료

==================================================

6️⃣ 코드 블록

● 한 줄 코드
`git pull origin develop`

● 여러 줄 코드

```bash
git pull origin develop
npm install
npm run dev
```

사용 가능한 언어 예시
`bash`, `js`, `ts`, `python`, `json`, `html`, `css`

==================================================

7️⃣ 인용문 (Quote)

> 이것은 인용문입니다  
> 여러 줄도 가능합니다

==================================================

8️⃣ 구분선 (Section Divider)

---

==================================================

9️⃣ 링크 (Link)

[구글](https://google.com)

==================================================

🔟 이미지 (Image)

![대체텍스트](이미지주소)

예시:
![CI/CD 흐름](./images/cicd.png)

==================================================

1️⃣1️⃣ 표 (Table)

| 항목 | 설명 |
|------|------|
| CI | 자동 테스트 |
| CD | 자동 배포 |

==================================================

1️⃣2️⃣ 이모지 (Emoji)

🔥 🚀 ✅ 📌 ⚠️  
(Windows: Win + .)

==================================================

1️⃣3️⃣ 접기 / 펼치기 (Details)

<details>
<summary>자세히 보기</summary>

내용 작성

</details>

==================================================

1️⃣4️⃣ 정렬 (HTML 혼용 가능)

<div align="center">
가운데 정렬
</div>

==================================================

1️⃣5️⃣ 이스케이프 (문법 그대로 출력)

\*별표 그대로 출력\*
\# 제목 그대로 출력
\- 목록 기호 그대로 출력

==================================================

📌 참고
- 마크다운 문법을 그대로 보이게 하려면 코드 블록 사용
- VS Code 미리보기: Ctrl + Shift + V
- 옆 미리보기: Ctrl + K → V
~~~
